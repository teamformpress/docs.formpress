Developers Guide

Learn how to contribute to FormPress and extend its functionality.

Thank you for considering contributing to FormPress. This guide explains how you can contribute to the project, report issues, and help with translations.

Reporting Issues

If you find a bug, please report it to our GitHub repository unless it is about a security vulnerability. Creating an issue is always recommended, but for minor issues like correction of a typo, you can create a pull request instead of an issue.

If you can fix an issue, create a pull request that links to the target issue.

Security Note: Security issues should not be discussed publicly. Please do not publish them through GitHub or other channels. For security vulnerabilities, it is recommended you use the Wordfence vulnerability submission form.

Before Reporting

Prior to reporting an issue, please:

  1. Double-check if the issue is caused purely due to a bug of the plugin's latest version.
  2. Confirm that the problem occurs even in a "vanilla" environment where only WordPress core, the default theme, and FormPress are activated.

Coding Standards

For WordPress plugin development, we follow strict coding standards. We ask you to respect these standards in your contributions. FormPress follows WordPress coding standards to ensure long-term maintainability.

The plugin avoids unnecessary dependencies and keeps features isolated. Key aspects include:

  • Adherence to WordPress Coding Standards.
  • Validation and sanitization of all inputs and outputs.
  • Clear and documented code structure.

Contributing with Translations

More than half of our users are non-English speakers. Maintaining translations for their languages is a great help.

We use translate.wordpress.org (GlotPress) to manage translations. This is a collaborative platform where thousands of volunteers participate.

Helping Other Users

Coding is not the only way to contribute. Helping other users is also a respected contribution.

Users often come to the official support forum. If you can answer their questions, please do so!
