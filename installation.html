Installation

Get up and running with FormPress in minutes.

Requirements

  • WordPress 6.4 or higher
  • PHP 7.4 or higher

Automatic Installation

  1. Log in to your WordPress admin dashboard.
  2. Navigate to Plugins > Add New.
  3. Search for "FormPress".
  4. Click Install Now and then Activate.

Manual Installation

  1. Download the plugin zip file from the WordPress Plugin Repository or the repository.
  2. Upload the `formpress` folder to the `/wp-content/plugins/` directory.
  3. Activate the plugin through the Plugins screen in WordPress.

Note: After activation, a new menu item "FormPress" will appear in your WordPress admin dashboard, where you can start creating forms.

Upgrading

FormPress adheres to WordPress plugin standards. You can update it directly from the Dashboard > Updates screen whenever a new version is released.
