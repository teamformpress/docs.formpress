Quick Start

  1. Install the plugin: Plugins → Add New → Search & Install "Formpress".
  2. Create a form: Forms → Add New. Add fields and save.
  3. Publish: copy the shortcode and paste into a page or use the Formpress block.
  4. Set up email under Settings → Notifications.

Need a clean website for docs? Enable GitHub Pages for teamformpress/docs.formpress.
